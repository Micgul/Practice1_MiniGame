 - 알고리즘
	성능을 최대한으로 살리는 문제 해결 방법
	ex) 길찾기 알고리즘, 맵 생성 알고리즘 등등.
	문제 해결이라는 것은 똑같지만 특정 상황에 따라서 알고리즘 간 성능차이가 발생한다.
	
	a와 b 변수의 내용들을 교체한다.
	
	
 - 디자인 패턴
	디자인 : '코드 구조'
	패턴 : 자주 반복되는 모양 

	성능이 목적이 아니라 코드 구조 설계 최적화의 목적 == 코드 가독성 및 설계 유연성
	설계 유연성 : 코드가 얼마나 자유롭게 변형될 수 있는지
	성능은 디자인패턴에서 그렇게 크게 중요하지 않다. == 큰 부담이 없는 이상 성능 이슈는 취급하지 않는다.
